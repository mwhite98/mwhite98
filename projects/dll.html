Rainy Day

Girlboss Lorem Ipsum

The prompt for this project was to generate a note using python, and insert that note into any website using the browser's dev console.

The website I chose to share my note in is the Government of Canada's unemployment insurance application webpage. For the note, I created a dictionary of words sourced from a numuber of girlboss motivational quote pages on Instagram. I implemented a simple syntax tree using python, and used this script to generate the content for the updated webpage. I re-styled the webpage using CSS.
